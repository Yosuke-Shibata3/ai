# 🏗️ クレーン解体 — Wrecking Ball Demolition

クレーンの鉄球を振り回して建物をぶっ壊す、ブラウザだけで遊べる物理パズルゲーム。

![type: HTML5 / Canvas / Matter.js](https://img.shields.io/badge/HTML5-Canvas-orange)

## 遊び方

`index.html` をブラウザで開くだけ（インストール不要）。

| 操作 | 内容 |
| --- | --- |
| 🖱️ **ドラッグ** | 鉄球をつかんで引っぱり、離してスイング |
| **A / D**（← →） | クレーン（トロリー）を左右に移動 |
| **W / S**（↑ ↓） | ワイヤーを伸ばす / 縮める |
| **Space** | 鉄球を真上に戻す |

## ルール

- 鉄球をぶつけてブロックを倒す・吹き飛ばすとスコア加算（1ブロック +100）
- 解体率が **92%** を超えるとクリア（+500 ボーナス）→ 次レベルへ
- レベルが上がるほど建物が高く・数が増える

## 技術

- 単一の `index.html`（依存はランタイムCDNの [Matter.js](https://brm.io/matter-js/) のみ）
- 物理演算: Matter.js / 描画: 自前の Canvas レンダラー（粉塵パーティクル・カメラシェイク付き）
